# Changelog

All notable changes to this project will be documented in this file.

## [0.3.0] - 2025-12-28

### Added
- **Next.js 16 Support**: Updated base template to Next.js 16.1.1 with Turbopack as default bundler

### Changed
- **Dependencies Updated**: All packages updated to latest wanted versions (patch/minor updates):
  - Next.js: 15.3.4 → 16.1.1
  - React & React DOM: 19.0.0 → 19.2.3
  - Prisma & @prisma/client: 6.10.1 → 6.19.1
  - Tailwind CSS: 4.0.x → 4.1.18
  - @tailwindcss/postcss: 4.0.x → 4.1.18
  - TypeScript: 5.x → 5.9.3
  - ESLint: 9.x → 9.39.2
  - @types/node: 20.x → 20.19.27
  - And other dependency updates

### Security
- Zero vulnerabilities detected across all dependencies

## [0.2.1] - 2024-XX-XX

### Initial Release
- CLI scaffolding tool for Next.js starter projects
- Base template with Next.js, Prisma, Tailwind CSS
- Optional authentication (Better Auth)
- Optional Mailgun integration
